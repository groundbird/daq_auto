# Observation Planning Tools

This repository contains two Python scripts for planning and executing astronomical observations:

1. `planner.py`: Generates observation timing schedules in UTC. Almost same as slack_reporter/astro/moon_reporter.py
2. `reset_scheduler.py`: Executes reset commands for equipment based on the generated schedule

## planner.py

A tool for generating observation timing schedules based on astronomical body positions. The script calculates when specific celestial bodies cross certain elevation angles and generates a schedule of actions.

### Features

- Calculates rising and setting times for celestial bodies
- Generates schedules for equipment restarts
- Includes sun avoidance calculations
- Supports various celestial bodies (moon, planets, stars)

### Usage

```bash
python planner.py <body> [--date DATE] [--sun SUN_ANGLE] [--line ELEVATION]

Arguments:
  body                  Astronomical body name (e.g., 'moon', 'jupiter')
  --date               Observation date (optional, default: current time)
  --sun, -s            Maximum sun elevation to avoid (default: 60 degrees)
  --line, -l           Elevation angle of the telescope (default: 70 degrees)
```

### Example

```bash
# Generate schedule for moon observation
./planner.py moon

# Generate schedule for Jupiter with custom parameters
./planner.py jupiter --date "2025-03-01" --line 65 --sun 55
```

## reset_scheduler.py

An automated scheduler that executes reset commands for observation equipment based on the timing schedule generated by `planner.py`. The script runs for a 24-hour period and includes automatic sky observation commands.

### Features

- Automated execution of reset commands
- Automatic sky observation after resets
- Logging of all operations
- User confirmation before execution
- Using screen is recommended for long-running operations

### Usage

```bash
./reset_scheduler.py [--target TARGET] [--elevation ELEVATION] [--sun-avoid SUN_ANGLE]

Arguments:
  --target             Target celestial body (default: moon)
  --elevation          Elevation angle for observations (default: 70 degrees)
  --sun-avoid         Sun avoidance angle (default: 60 degrees)
```

### Running with Screen

For long-term operation, it's recommended to run the scheduler in a screen session:

```bash
# Create new screen session
screen -R auto

# Start the scheduler
./reset_scheduler.py --target moon

# Detach from screen (keep running in background)
# Press Ctrl+A, then D

# Later, to reattach to the session
screen -r auto

# List all screen sessions
screen -ls

# Kill the screen session when done
screen -X -S auto quit
```

## Notes

- Both scripts assume the observatory location is set to Observatorio del Teide
- The scheduler changes the target name to sky every 20 minutes with every reset command.
- Time calculations are in UTC
